**The shared engineering standard for every [`@astronordic`](https://github.com/astronordic) repository.**

_O padrão de engenharia compartilhado por todos os repositórios [`@astronordic`](https://github.com/astronordic)._

## 🇬🇧 English

This is a **special repository**. On GitHub, a public repository named `.github` provides
**default community health files** — `CONTRIBUTING`, `CODE_OF_CONDUCT`, `SECURITY`, issue &
pull-request templates — to **every other repository on the account that doesn't define its own**.

In other words: this repo is the **single source of truth**. Define a convention here once, and it
is *necessarily followed* everywhere. Individual repos may override a file when they genuinely need to.

### What lives here

| File | Purpose |
|------|---------|
| [`STANDARD.md`](./STANDARD.md) | The full engineering standard (commits, branches, READMEs, versioning, licensing) |
| [`CONTRIBUTING.md`](./CONTRIBUTING.md) | How to contribute across all repos |
| [`CODE_OF_CONDUCT.md`](./CODE_OF_CONDUCT.md) | Community expectations |
| [`SECURITY.md`](./SECURITY.md) | How to report a vulnerability |
| [`.github/ISSUE_TEMPLATE/`](./.github/ISSUE_TEMPLATE) | Bug report & feature request forms |
| [`.github/PULL_REQUEST_TEMPLATE.md`](./.github/PULL_REQUEST_TEMPLATE.md) | The PR checklist |
| [`templates/`](./templates) | Drop-in files: `LICENSE`, `.editorconfig`, CI workflows, Dependabot |

### The projects that follow it

## 🇧🇷 Português

Este é um **repositório especial**. No GitHub, um repositório público chamado `.github` fornece
**arquivos-padrão de comunidade** — `CONTRIBUTING`, `CODE_OF_CONDUCT`, `SECURITY`, templates de issue
e de pull request — para **todos os outros repositórios da conta que não definirem os seus próprios**.

Ou seja: este repo é a **fonte única da verdade**. Você define uma convenção aqui uma vez, e ela é
*necessariamente seguida* em todo lugar. Um repo específico pode sobrescrever um arquivo quando
realmente precisar.

### O que mora aqui

| Arquivo | Para quê |
|---------|----------|
| [`PADRAO.md`](./PADRAO.md) | O padrão de engenharia completo (commits, branches, READMEs, versão, licença) |
| [`CONTRIBUTING.md`](./CONTRIBUTING.md) | Como contribuir em todos os repos |
| [`CODE_OF_CONDUCT.md`](./CODE_OF_CONDUCT.md) | Expectativas de convivência |
| [`SECURITY.md`](./SECURITY.md) | Como reportar uma vulnerabilidade |
| [`templates/`](./templates) | Arquivos prontos: `LICENSE`, `.editorconfig`, workflows de CI, Dependabot |
